# Dove Nest Goa Website

Final animated mobile-first website for Dove Nest, Santa Cruz Bandh, Goa.

Includes:
- Fixed dark mode background switching
- Dark blue + Goan green colour palette
- Mobile-first hamburger menu
- Theme changer button
- Animated floating transparent PNGs: Goan palm, pool, food, music
- WhatsApp booking enquiry flow
- SEO metadata and schema
- Gallery, booking, contact, feedback, stay & pool pages

Open `index.html` to view the website.
